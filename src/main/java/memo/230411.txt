백준 활용


기본 자료형(Primitive type) : int, long, double, char, boolean, ...
 - stack에 값이 들어감

참조 자료형(Reference type) : 배열, String처럼 대문자로 시작하는 것들
 - stack에는 변수만(heap 영역의 주소 참조), heap영역에 데이터가 있다.

래퍼 클래스(Wrapper Class or Boxed Class) : 참조자료형임
Integer, Long, Double, Character, Boolean 처럼 기본 자료형을 클래스화 시켜놓음 대문자로 시작함



변수 하나만 저장가능
배열 여러개 가능
클래스 필드, 메소드로 저장가능


자료구조가 중요함 - 서점이나 그런 곳에서 자료구조에 관한 책 알아보기
리스트 셋 큐에 관한 거를 유튜브나 그런 곳에서 찾아보기
콜렉션 자료구조
- List
- Set
- Queue
