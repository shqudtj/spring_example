if만 적은 코드는 다 별개의 조건문이므로 하나가 참이라도 다음 것도 실행함

else if 를 사용하면 조건문들 중에 하나가 참이라면 조건문 탈출


switch-case 문은
하나가 참이면 탈출이 아니라 다음 조건도 실행을 함
이를 방지하려면 break; 를 실행문 다음에 적는다

case문에서 조건의 변수가 여러가지가 같다면
case 1 :
case 2 :
case 3 :
syso @@@
break;
case 4 : case 5 : case : 6
syso @@@
break;
라고 써도 된다
