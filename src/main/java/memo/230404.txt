클래스 수업
클래스를 만들어서 다른 패키지에서 복사를 해서 그대로 붙일 시 import가 생기는데 이를 없애줘야함
이유는 default가 생략되어 적혀있기 때문에
default는 자료형 (String 등) 앞에 생략되어 적혀있음 -> 이는 같은 패키지 내에서만 접근 가능


이를 방지하려면 public을 적어야함
이러면 모든 패키지에서 가능함

앞으로 필드를 만들때 자료형 앞에 private를 적어줘야함
